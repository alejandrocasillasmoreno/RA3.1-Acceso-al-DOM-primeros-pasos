# RA3.1-Acceso-al-DOM-primeros-pasos
Descripción:
Este taller se centra en dar vida a páginas web.Para ello usaremos JavaScript(JS)
para manipular la estructura de la página (DOM)

Donde aprenderemos los elementos basicos de JS para modificar DOM.

Objetivo de la actividad
El objetivo de esta actividad es dominar la programación del lado del cliente para ello
usaremos la manipulación del DOM, el manejo de eventos,Integrar datos externos

Tecnologías usadas:
-HTM5
-JavaScript(ES6+)
-Bootstrap 5
-JSON Placeholder API
